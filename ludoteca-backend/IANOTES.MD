# IA_NOTES.md

## Uso de Inteligencia Artificial en el proyecto

Durante el desarrollo de este sistema de **Ludoteca (Node.js + TypeScript + PostgreSQL)** se utilizó asistencia de inteligencia artificial como apoyo técnico y de documentación.

La IA fue empleada principalmente para:

* Explicación de conceptos de backend (JWT, bcrypt, middlewares, rutas, controladores).
* Generación de ejemplos de código base y estructuras CRUD.
* Creación de ejemplos de archivos XML para importación masiva.
* Elaboración de descripciones técnicas para documentación y README.
* Apoyo en la construcción de diagramas UML conceptuales.
* Resolución de errores comunes de configuración en Express DURANTE EL LEVANTAMIENTO.



## Alcance del uso de IA

La inteligencia artificial **no reemplazó la implementación**, sino que fue utilizada como:

* Herramienta de aprendizaje.
* Soporte de depuración.
* Asistente de documentación técnica.

Toda la integración, pruebas, ejecución del sistema, conexión a base de datos, autenticación y funcionamiento real del backend fueron **configurados y verificados manualmente**.
